# Master Baja Bangunan

A modern and responsive website for **Master Baja Bangunan**, a building materials business providing construction products and information for customers.

## About the Project

This project was developed as a responsive business website using HTML, CSS, and JavaScript.

The website provides information about building materials, product pricing, business information, and an easy way for customers to access the product brochure.

## Features

- Responsive website design
- Product and pricing section
- Downloadable product brochure
- WhatsApp contact integration
- Smooth animations
- Mobile-friendly interface
- Modern navigation
- Product images and information

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AOS (Animate On Scroll)
- Git
- GitHub

## Project Structure

- `index.html` — Main website page
- `css/` — Website styles
- `js/` — JavaScript files
- `images/` — Website images and product assets
- `assets/` — Additional project resources and brochure
- `.gitignore` — Files ignored by Git

## Purpose

This project was created to build a professional online presence for Master Baja Bangunan and make it easier for customers to explore products and access business information.
